## 3. Responsive Design Considerations

### Breakpoints
- **Desktop:** 1280px+ (full layout)
- **Tablet:** 768px - 1279px (collapsed sidebar, stacked panels)
- **Mobile:** < 768px (bottom nav, single column)

### Mobile-Specific Adaptations
- Bottom navigation bar replaces sidebar
- Wizard steps become full-screen
- Editor shows one panel at a time (toggle between preview/settings)
- Simplified timeline for scene selection
